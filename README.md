в Җ                          к’· пё¶  рқ“јрқ“ұрқ“®  вӮ’бөЈв Җрқ“ұрқ“®рқ“»   в—ҹ рқ“ҳ вҷЎ рқ“ҷрқ“Ірқ“·рқ“° рқ“Ёрқ“ҫрқ“Әрқ“·  р–№ӯ  Нң    вҖј

рқ——рқ—ј рқ—»рқ—јрқҳҒ рқ—°рқ—јрқ—ҪрқҳҶ рқ—јрқ—ҝ рқ—¶рқ—»рқҳҖрқ—Ҫрқ—ј рқ—®рқ—»рқҳҶ рқ—јрқ—і рқ—әрқҳҶ рқ—Ҫрқ—јрқ—»рқ—¶рқ—ІрқҳҖ/рқҳҖрқ—ёрқ—¶рқ—»рқҳҖ. рқ—ңрқ—і рқҳҶрқ—јрқҳӮ рқҳ„рқ—®рқ—»рқ—»рқ—®, рқ—®рқҳҖрқ—ё рқ—Ҫрқ—Ірқ—ҝрқ—әрқ—¶рқҳҖрқҳҖрқ—¶рқ—јрқ—» рқ—ірқ—¶рқ—ҝрқҳҖрқҳҒ.

рқ—Јрқ—№рқҳҖ рқ—ұрқ—јрқ—»'рқҳҒ рқ—Ҝрқ—І рқҳҖрқ—өрқҳҶ рқҳҒрқ—ј c+рқ—ө рқ—әрқ—І рқ—јрқ—ҝ рқ—¶рқ—»рқҳҒ рқҳ„рқ—¶рқҳҒрқ—ө рқ—әрқ—І (рқҳӮрқ—»рқ—№рқ—ІрқҳҖрқҳҖ рқҳҒрқ—өрқ—Ірқ—ҝрқ—І'рқҳҖ рқ—ұрқ—»рқ—¶ рқ—јрқ—» рқ—әрқҳҶ рқ—»рқ—®рқ—әрқ—І) рқ—Јрқ—№рқҳҖ рқҳҒрқ—®рқ—№рқ—ё/рқ—¶рқ—»рқҳҒ рқ—ірқ—¶рқ—ҝрқҳҖрқҳҒ рқҳҒрқ—өрқ—јрқҳӮрқ—ҙрқ—ө рқ—ң'рқ—ә рқҳҖрқ—өрқҳҶ (вҲ©ВҙвҲҖ`вҲ©)

рқ—ң рқҳҖрқ—өрқ—¶рқ—Ҫ рқ—әрқҳҶ рқ—јрқ—° рқҳ„рқ—¶рқҳҒрқ—ө рқ—қрқ—¶рқ—»рқ—ҙ рқ—¬рқҳӮрқ—®рқ—» (рқ—өрқҳҖрқ—ҝ) вҖ”рқ—јрқ—°рқҳ…рқ—°рқ—®рқ—»рқ—јрқ—»/рқҳҶрқҳӮрқ—әрқ—І рқ—®рқҳҒрқ—Ҫ.
